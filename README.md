# SF-DeepLink: Salesforce Setup Shortcut Extension

**SF-DeepLink** is a browser extension designed for Salesforce Developers and Admins to bypass the "Setup" click-fatigue. Access Flows, Public Groups, Apex Classes, and Profiles with a single click or keyboard shortcut.

## 🚀 Features
* **Direct Access:** Jump straight to Flow Builder, Profiles, or Permission Sets.
* **Context Aware:** Automatically detects your current Salesforce Org domain.
* **Quick Search:** Filter through common Setup items without waiting for the Setup sidebar to load.
* **Custom Shortcuts:** Map your most-used metadata types to specific keys.

## 🛠 Installation
1. Clone this repository: `git clone https://github.com/your-username/sf-deeplink.git`
2. Open Chrome/Edge and navigate to `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the project folder.

## 📂 Project Structure
* `manifest.json`: Extension configuration.
* `popup.html`: The UI for the shortcut menu.
* `scripts/`: Logic for URL generation and redirection.
